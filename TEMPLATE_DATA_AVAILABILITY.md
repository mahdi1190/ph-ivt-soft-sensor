# Data & Software Availability Statement (for the manuscript)

**Data.** All data underlying the findings of this study will be made available via the GitHub repository at: https://github.com/<org-or-user>/<repo> and via a preserved snapshot on Zenodo (DOI to be added on acceptance). The archive will contain: (i) raw input datasets (D1–Dk), (ii) processed data required to recreate the figures and tables in the manuscript, and (iii) checksums and metadata. See `DATA.md` for a complete inventory and licenses.

**Code.** The analysis code, including Pyomo models and scripts to regenerate all results, is available at https://github.com/<org-or-user>/<repo> under an MIT License. The exact version used for the paper will be archived at Zenodo (DOI above) and tagged as `v1.0.0` in the repository.

**Software & dependencies.** A reproducible environment is specified in `environment.yml` and `requirements.txt`. Instructions to recreate the environment and reproduce the main figures are provided in the repository `README.md`.

**Corresponding author.** Zoltan Kis (z.kis@sheffield.ac.uk).

**Restrictions.** If any dataset cannot be publicly shared (e.g., due to third‑party licensing), `DATA.md` specifies the legal basis and provides a concrete route for qualified access (e.g., direct request to rightsholder, or controlled-access repository).

**Editors/Reviewers access to proprietary tools (if any).** In line with *Digital Discovery*'s **Software and Code** guidance, instructions for editor/reviewer access to any proprietary components (if used) are documented in `DATA.md` under “Editors’ access”.
